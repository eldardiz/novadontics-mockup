# Novadontics — Pricing & Brand Mockup

Pitch mockup for Novadontics — a redesigned pricing page and a refreshed brand identity styleguide.

## Live

- `/` — Pricing page
- `/styleguide` — Brand identity styleguide (standalone, no nav back)

## Stack

Plain HTML / CSS / JS. No build step. Deployed as static to Vercel.

## Design system

Warm editorial palette + Instrument Serif × Inter pairing. See `/styleguide` for the full system.

| Token       | Hex       | Role                                     |
| ----------- | --------- | ---------------------------------------- |
| Cream       | `#FAF5EB` | Page background                          |
| Paper       | `#FFFEFA` | Card surfaces                            |
| Sand        | `#EDE3D2` | Dividers, table rows                     |
| Ochre       | `#E27D2A` | Primary CTA, checkmarks, italic accents  |
| Honey       | `#F2B53A` | "Most Popular" pill, dark-section italic |
| Espresso    | `#2B1D14` | Dark sections, headings                  |
| Bark        | `#5C4A3D` | Body text                                |
| Driftwood   | `#8B7A6B` | Secondary text                           |
